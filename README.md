# Donic

Arduino project for sensing distance and giving feedback to the user.

## Configuration

### Ultrasonic sensor

- Echo on pin 11
- Trig on pin 12

### LCD

- SDA on pin A4
- SCL on pin A5

### Joystick

- VRx on pin A0
- VRy on pin A1
- SW on pin 2

### Servo motor

- pin 3 

### Buzzer

- pin 9
